- `icon_story.png` - 故事图标（未选中状态）
- `icon_story_selected.png` - 故事图标（选中状态）
- `icon_around.png` - 身边图标（未选中状态）
- `icon_around_selected.png` - 身边图标（选中状态）
- `icon_activity.png` - 活动图标（未选中状态）
- `icon_activity_selected.png` - 活动图标（选中状态）

## 其他图片资源
- `banner.jpg` - 首页顶部Banner图

- 图标尺寸：48x48px 或 64x64px
- Banner图：建议尺寸 750x300px
- 其他图片：根据实际需求调整，但建议使用WebP格式以优化加载速度
