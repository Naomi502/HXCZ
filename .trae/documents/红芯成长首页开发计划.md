# 红芯成长首页设计方案

## 一、首页结构设计

### 1. 数据统计卡片区域
展示核心业务数据概览:
- 活动总数 (统计所有活动)
- 进行中活动 (status=1的活动)
- 学生总数 (HxStudentProfile记录数)
- 积分发放总数 (HxPointLog累计积分)
- 故事总数 (HxStory记录数)
- 本周活动参与人数

### 2. 图表展示区域
- **活动趋势图**: 近6个月活动发布数量趋势
- **积分增长图**: 近6个月积分发放趋势
- **故事阅读排行**: Top 10 热门故事

### 3. 快捷操作区域
- 发布活动 (跳转到活动管理页面)
- 审核活动证明 (跳转到活动参与审核页面)
- 发布故事 (跳转到故事管理页面)
- 积分调整 (跳转到积分管理页面)

### 4. 最新动态区域
- 最新活动列表 (最近发布的活动)
- 最新故事列表 (最近发布的故事)
- 最新积分变动 (最近7天的积分流水)

### 5. 操作说明文档
- 系统功能介绍
- 各模块操作指南
- 常见问题解答

## 二、技术实现要点

### 前端实现
1. 创建新的首页组件,替换现有的若依默认首页
2. 使用 Element UI 的卡片、表格、图表组件
3. 复用现有的图表组件 (LineChart, BarChart, PieChart)
4. 使用 CountTo 组件实现数字动画效果

### 后端接口
1. 新增统计数据接口 `/hx/dashboard/statistics`
2. 返回各类统计数据和图表数据
3. 使用现有的业务接口获取列表数据

### API设计
```javascript
// 新增API文件: src/api/hx/dashboard.js
export function getDashboardData() {
  return request({
    url: '/hx/dashboard/statistics',
    method: 'get'
  })
}
```

4. **新增**: `ruoyi-system/src/main/java/com/ruoyi/system/service/IHxDashboardService.java` - 统计服务接口
