# 数字人悬浮窗与界面优化方案

我将为您实现一个“全息投影”风格的数字人悬浮窗，利用 CSS 滤镜技术去除视频黑底，并移除原有的 AI 头像。

## 1. 移除 AI 气泡头像
- **文件**: `qa.wxml`
- **操作**: 删除 AI 消息列表中的 `<image class="avatar" ...>` 标签。
- **样式**: 修改 `qa.wxss`，去除 AI 消息气泡的左侧留白，使其自然靠左对齐。

## 2. 添加数字人悬浮窗 (无缝切换)
- **文件**: `qa.wxml`
- **实现**:
  - 在页面顶部添加 `.digital-human` 悬浮容器。
  - **双视频预加载策略**：同时渲染“说话”和“待机”两个 `<video>` 组件。
    - **说话视频**: `/images/speak.mp4` (当 `isSpeaking` 为 true 时显示)
    - **待机视频**: `/images/sleep.mp4` (当 `isSpeaking` 为 false 时显示)
  - 使用 `hidden` 属性控制显示/隐藏，确保切换时**零延迟、不黑屏**。
  - 均设置为 `autoplay`、`loop`、`muted` (静音)，因为声音由 TTS 播放，视频仅负责视觉动效。

## 3. 视觉去黑底 (黑科技)
- **文件**: `qa.wxss`
- **技术**: 使用 `mix-blend-mode: screen;` (滤色模式)。
- **效果**: 视频中的黑色背景将变为透明，仅保留发光的数字人主体，使其像全息投影一样悬浮在浅色背景上。
